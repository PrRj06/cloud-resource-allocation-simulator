# CloudOptima

CloudOptima is now structured as a static Vercel-ready site:

- `index.html` for the UI structure
- `styles.css` for all presentation rules
- `script.js` for simulator behavior and chart logic
- `vercel.json` for client-side routing fallback

## Local run

Open `index.html` directly in a browser or serve the folder with any static server.

## Deploy to Vercel

1. Push the repository to GitHub.
2. Import the repo into Vercel.
3. Deploy with the default settings. No build step is required.
